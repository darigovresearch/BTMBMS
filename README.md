# BTMBMS
A Better Than Manual Battery Management System

## Installation
- Clone the repository
- Install the requirements 
```
  pip install -r requirements.txt
```

## Run the app
- Go to the `App` directory
- Run the app

```
  python app.py
```
- Go to http://localhost:8766/
